Projects

Networked Chatroom

A real-time chatroom application built using ImGui for the user interface and Winsock for robust network communication. The project features a client-server architecture supporting multiple simultaneous connections, direct messaging, and broadcast messaging with graceful connection and disconnection handling.

  • Client-Server Architecture
  • Direct & Broadcast Messaging
  • Graceful Connection and Disconnection Handling
  • Real-time UI with ImGui

Technologies: C++, Winsock, ImGui, Multi-threading
